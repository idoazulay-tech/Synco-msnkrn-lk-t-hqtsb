ARCHITECTURAL PRINCIPLES

1. One meaningful user activity may produce multiple distinct factual derived memories.

- task_completion
- actual_duration
- planned_vs_actual_duration

"one LearningEvent = one memory only."

3. Correct duplicate rule:
The same derived memory must never be created twice from the same factual source.

4. Correct deterministic identity:
sourceLearningEventId + memoryKind + schemaVersion

DERIVATION RULES

1. task_created
Produce exactly:

Text:
"המשתמש יצר משימה חדשה: {title}."

Conditions:
- Only if taskTitleSnapshot exists and is non-empty.
- Do not infer importance, habit, emotion, or intention beyond the factual creation.

2. task_completed
Produce exactly:
- memoryKind: 'task_completion'

Text:
"המשתמש השלים את המשימה: {title}."

Conditions:
- Only if taskTitleSnapshot exists and is non-empty.
- Do not include duration.

3. task_execution_completed

Only if duration evidence is accepted:
- actualStartSource === 'execution_start'
  OR actualStartSource === 'user_reported'
- actualDurationMinutes is a valid positive finite number

Produce:
A. memoryKind: 'actual_duration'

For execution_start text:
"המשתמש ביצע את המשימה: {title}. משך הביצוע בפועל שאושר באמצעות טיימר: {actualDurationMinutes} דקות."

For user_reported text:
"המשתמש ביצע את המשימה: {title}. משך הביצוע בפועל שדווח ואושר על ידי המשתמש: {actualDurationMinutes} דקות."

B. memoryKind: 'planned_vs_actual_duration'
Only if plannedDurationMinutes is also a valid positive finite number.

Text:
"המשימה {title} תוכננה למשך {plannedDurationMinutes} דקות ובוצעה בפועל במשך {actualDurationMinutes} דקות, לפי זמן שאושר."

Metadata must preserve the evidence source:
For execution_start:
- evidenceType: 'timer_confirmed'
- durationConfidence: 'confirmed'

For user_reported:
- evidenceType: 'user_reported_duration'
- durationConfidence: 'confirmed'

Do NOT generate duration-derived memories for:
- planned_fallback
- expired_execution_start_fallback
- user_reported_no_duration
- missing or unknown source
- null, zero, negative, or invalid actual duration

Do NOT generate any pattern or recommendation.
