Synco must not connect memories to the current situation as fact unless confidence is strong.

Task:
Replace the single hardcoded threshold behavior with a confidence-tier policy.

Implement these levels:

1. ignore
score < 0.35
Do not use the memory.

2. weak_signal
score >= 0.35 and score < 0.50
Use only as a signal to ask a clarifying question.
Do not present it as a fact about the user.

3. suggested_context
score >= 0.50 and score < 0.70
Use as a possible connection.
The assistant may say "יכול להיות שזה קשור ל..." and ask for confirmation.

4. active_context
score >= 0.70
Use as active context.
Still do not perform high-impact actions without user confirmation.
